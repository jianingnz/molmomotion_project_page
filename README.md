# MolmoMotion Project Page

Project page for **MolmoMotion: Language-guided 3D motion forecasting**, reusing the design system from the MolmoMotion blog post (Manrope typeface, AI2 palette, dark-green hero, cream body).

## Structure
- `index.html` — the page (hero with authors/links + article body)
- `styles.css` — shared blog styling plus project-page hero extras (authors, affiliations)
- `images/` — figures, demo clips, graphs

## Status
Scaffold with placeholders (authors, links, abstract, teaser). Content to be filled in.

```
python3 -m http.server 8000
```
then visit http://localhost:8000
